#自学指南

一、LLM 基础 & Prompt 工程（必学，所有 AI 项目底层）
1. 中文免费
DeepSeek 提示词工程指南（极简，适合新手）
https://www.deepseek.com/prompt-guide
DeeepSeek提示词库
https://api-docs.deepseek.com/prompt-library
学：温度、Token、上下文、系统提示词、格式约束（你代码里的 system_prompt 就是这个）
李沐・动手学深度学习（LLM 极简版）
只看「大模型基础」章节，不用学训练，懂原理即可
2. 英文权威（面试加分，岗位要求懂海外生态）
Prompt Engineering Guide（全球最经典免费）
https://learnprompting.org/
学：零样本 / 少样本、格式输出、Agent 专用 Prompt 写法
二、Agent 智能体核心（ReAct、工具调用、记忆，你现在的核心）
1. 中文入门（直接看懂你写的 Baby Agent）
B 站：Agent 从 0 到 1 极简教程（强推）
关键词搜：ReAct Agent 从零手写
重点看：思考 - 行动 - 观察循环、工具调用逻辑，和你代码完全对应
LangChain 中文文档（新手只看入门部分）
https://python.langchain.com.cn/docs/introduction/
只学：Agent、Tools、Memory（记忆）3 个模块，其他跳过
2. 英文官方（岗位必备，懂海外 Agent 生态）
LangChain 官方 Agent 教程
https://python.langchain.com/docs/modules/agents/
全球最主流 Agent 框架，面试必问
Baby‑Agent 官方极简文档
https://github.com/baby-llm/baby-agent
几十行代码讲清 Agent 本质，和你项目完全匹配
三、RAG 检索增强（求职必考，必须吃透）
1. 中文保姆级
硅基流动 RAG 入门教程
https://docs.siliconflow.cn/docs/rag/intro
学：文档切片、向量 Embedding、向量库、检索重排
B 站：RAG 从零搭建本地知识库
关键词：极简RAG Python，跟着敲代码做第二个作品集
2. 英文权威
LlamaIndex 官方文档（RAG 最主流框架）
https://docs.llamaindex.ai/
企业 90% 私有 AI 都用这个，学会直接加分
四、工作流 Workflow & 多智能体（岗位核心能力）
1. 可视化低代码（最快上手）
Dify 官方教程（中文）
https://docs.dify.ai/zh-hans/
拖拽搭建 Agent 工作流，不用写大量代码，完美匹配 Vibe Coding
FlowiseAI 可视化 Agent 教程
零代码拼接 RAG+Agent + 工具，理解流程逻辑
2. 代码级多智能体
CrewAI 官方示例（中文）
https://github.com/joaomdmoura/crewAI-examples
多角色 Agent 协作，做第三个炫酷作品集
五、Vibe Coding & AI 编程工具（直接对应你投递的岗位）
1. 英文官方指南（岗位要求精通）
How to Vibe Code 2026 完整新手指南
学：Cursor/Trea/Windsurf 使用技巧、AI 写代码迭代逻辑
2. 实操教程
B 站搜：Cursor AI编程实战，学用 AI 快速开发 Agent 项目
六、Python 极简基础（只学必须的，不用全学）
菜鸟教程 Python 极简版
只看：函数、字典、列表、正则、requests 网络请求
完全够用你所有 Agent、RAG 项目
给你【14 天极简学习顺序】（照着走直接达标岗位）
1-3 天：LLM 基础 + Prompt 工程 → 吃透温度、Token、格式约束
4-6 天：Agent ReAct 原理 + 工具调用 → 优化你现有的 Baby Agent
7-10 天：RAG 向量知识库 → 完成第二个作品集
11-12 天：多智能体 CrewAI → 第三个作品集
13-14 天：Dify 工作流 + 海外模型体验 → 完善 GitHub、匹配 JD 全部要求
